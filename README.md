# Swimming Materials Advertising Website

This is a React.js frontend project styled with TailwindCSS for advertising swimming materials online. The UI is modern and engaging, designed to showcase products effectively.

## Setup Instructions

1. Install dependencies
```
npm install
```

2. Run the development server
```
npm start
```

3. Build for production
```
npm run build
```

## Technologies Used

- React.js
- TailwindCSS

## Features

- Responsive design
- Product listing with images and descriptions
- Modern UI with TailwindCSS

Feel free to customize and extend this project as needed.
